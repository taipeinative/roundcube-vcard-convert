Roundcube Webmail Contacts File Fix

This is the program by @taipeinative which aims to convert exported contacts from Roundcube Webmail into vCard 3.0 standard format.

In order to convert the file, please upload file by clicking the “Open” botton below and choose contact.vcf, which is the contacts file generated by Roundcube Webmail. Next, by clicking the “Convert” botton, the program will automatically download converted-contacts file.
